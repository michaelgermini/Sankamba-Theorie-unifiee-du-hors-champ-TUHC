# Chapitre 5 : Axe ontologique — Hors-champ fondamental

Ce chapitre traite du niveau le plus profond de la TUHC : le **hors-champ ontologique**.  
Il ne s’agit plus seulement de ce qui est hors du cadre perceptif ou social, mais de ce qui, dans la structure même de l’espace et des systèmes, **reste en-dessous du champ phénoménal** tout en le rendant possible. MULS fournit ici le langage privilégié pour décrire ce hors-champ fondamental.

## 5.1 L’espace comme relation primaire

### 5.1.1 Rupture avec l’espace-contenant

Dans l’intuition ordinaire, l’espace est souvent pensé comme un **contenant vide** dans lequel viennent se loger des objets.  
L’axe ontologique de Sankamba part d’une hypothèse différente :

- l’espace n’est pas d’abord un volume à remplir,  
- mais un **tissu de relations primaires** entre entités, événements, couches.

Autrement dit, ce qui existe n’existe pas “dans” un espace préalable :  
ce sont les **relations** elles-mêmes qui **font émerger** ce que nous appelons espace.

### 5.1.2 Relations visibles et invisibles

Si l’espace est relationnel, alors :

- certaines relations sont **manifeste** : trajectoires, distances mesurables, interactions observables ;  
- d’autres restent **hors-champ** : connexions structurelles, contraintes globales, dépendances non apparentes.

Le hors-champ ontologique désigne précisément ce niveau de **relations invisibles** qui conditionnent :

- ce qui peut apparaître,  
- comment cela peut apparaître,  
- et quelles transformations sont possibles.

### 5.1.3 Le hors-champ comme structure relationnelle

Dans ce cadre, le hors-champ n’est plus un “ailleurs” spatial, mais :

- une **partie de la structure relationnelle** qui n’est pas directement accessible à un observateur donné,  
- un **fond** qui soutient le champ, au sens où le tissu relationnel sous-jacent détermine ce qui peut se manifester dans le champ.

Sankamba propose ainsi une lecture du hors-champ comme **ontologie relationnelle** : ce qui ne se voit pas, mais qui définit ce que “voir” signifie dans un univers donné.

## 5.2 Morphs, liens structurels et couches

### 5.2.1 Morphs : briques de structure

Dans MULS, les **morphs** sont les **briques élémentaires** de description d’un mini-univers (S, B, C, E, X, T, etc.).  
Ils représentent des entités ou motifs structurels :

- **S** : cordes ou segments élémentaires,  
- **B** : branes ou surfaces de dimension supérieure,  
- **C** : dimensions compactifiées,  
- **E** : liens d’intrication,  
- **X** : interactions multiples,  
- **T** : topologies globales.

Dans la TUHC, ces morphs peuvent être interprétés comme les **unités minimales** du hors-champ ontologique :  
ce à partir de quoi l’espace, le temps et les interactions observables vont émerger.

### 5.2.2 Liens structurels

Les **liens structurels** décrivent comment les morphs sont connectés :

- quelles cordes sont reliées à quelles branes,  
- quelles dimensions sont compactifiées sur quelles structures,  
- quelles entités sont intriquées, etc.

Ces liens sont, par nature, **hors-champ** pour un observateur qui ne perçoit que les effets macroscopiques (forces, trajectoires, champs).  
Pourtant, c’est la **configuration de ces liens** qui détermine :

- la géométrie effective,  
- les canaux de propagation,  
- les régimes d’interaction possibles.

### 5.2.3 Couches (layers)

Les **couches** ou **layers** en MULS permettent de représenter :

- des niveaux distincts de structure,  
- des régimes différents (par exemple, une couche “micro” et une couche “macro”),  
- ou des sous-univers emboîtés.

Dans la TUHC, les couches sont un outil clé pour penser les **différents plans de hors-champ** :

- une couche peut être **visible** pour un type d’observateur et **hors-champ** pour un autre,  
- des phénomènes peuvent **traverser plusieurs couches** sans jamais être intégralement visibles sur une seule.

Le hors-champ ontologique se déploie donc **en profondeur** (dans les couches) et **en largeur** (dans les liens entre morphs).

## 5.3 Intrication et émergence

### 5.3.1 Intrication de structures

L’**intrication** (modélisée par les morphs E dans MULS) désigne des situations où :

- deux ou plusieurs entités ne peuvent pas être décrites de manière indépendante,  
- leur état, leur comportement, leur existence même sont **corrélés structurellement**.

Dans le cadre de la TUHC, l’intrication est une **figure paradigmatique du hors-champ** :

- ce qui relie deux éléments peut être totalement hors du champ de l’observation directe,  
- pourtant, les corrélations produites sont mesurables dans le champ.

Le hors-champ ontologique apparaît alors comme un **niveau d’intrication généralisée** entre éléments, couches et régimes.

### 5.3.2 Phénomènes émergents

Un phénomène est dit **émergent** lorsqu’il :

- ne se réduit pas à la somme des comportements de ses constituants,  
- mais résulte de la **configuration globale** des relations entre eux.

Dans MULS, l’émergence peut être décrite par :

- des configurations particulières de morphs et de liens,  
- des signatures d’univers (SIG) qui encodent des **invariants globaux** (comme la caractéristique d’Euler).

Dans la TUHC, un phénomène émergent visible (par exemple, un comportement collectif, un motif macroscopique) est l’**expression dans le champ** d’un **hors-champ structurel** complexe :

- ce sont les relations profondes, intriquées, multi-couches qui “produisent” le phénomène,  
- alors que seules ses manifestations superficielles sont observables.

### 5.3.3 Hors-champ comme origine de l’émergent

On peut ainsi dire que :

- le hors-champ ontologique est le **lieu de préparation** des phénomènes émergents,  
- l’émergence est la manière dont le hors-champ **remonte** dans le champ.

Sankamba se situe précisément dans cette tension :  
penser ce qui, en étant maintenu hors-champ, **organise** néanmoins toute apparition.

## 5.4 Formalisation via MULS

### 5.4.1 Signatures d’univers et hors-champ

MULS propose de caractériser un mini-univers par une **signature** :


SIG(Ns, Nb, Nc, Deff, χ)


où :

- \(Ns\) = nombre de cordes,  
- \(Nb\) = nombre de branes,  
- \(Nc\) = nombre de dimensions compactifiées,  
- \(Deff\) = dimensions effectives,  
- \(χ\) = caractéristique topologique globale.

Dans la TUHC, cette signature peut être lue comme une **signature de hors-champ ontologique** :  
elle encode ce qui, dans la structure de l’univers, **ne sera jamais directement visible** pour un observateur interne, mais conditionne tout ce qu’il percevra.

### 5.4.2 Mini-hors-champ ontologique (HCH-ONT)

On peut définir, par analogie avec MULS, un **mini-hors-champ ontologique** :


HCH_ONT {
  SIG(Ns, Nb, Nc, Deff, χ)
  // Morphs et liens non observables directement,
  // mais qui déterminent la dynamique apparente.
}


Ce HCH_ONT peut ensuite être relié à des **situations concrètes** (chapitres 6 et 7) :

- patterns de champ de forces,  
- régimes de propagation d’information,  
- possibilités de configuration d’un univers social ou technique.

### 5.4.3 Intégration TUHC–MULS

L’intégration TUHC–MULS repose sur quelques principes :

- à chaque **univers phénoménal** étudié (social, esthétique, technique), on peut associer un ou plusieurs **mini-univers MULS** qui en représentent la structure profonde ;  
- ces mini-univers MULS constituent le **hors-champ ontologique** de la situation ;  
- les variations de structure dans MULS correspondent à des **transformations de hors-champ** qui se traduisent, dans le champ, par des changements de comportements, de formes, de régimes.

Les RFC ultérieures (TUHC-0001, TUHC-0002) développeront cette intégration en détail, en proposant des **grammaires combinées** TUHC–MULS.

## 5.5 Exemples de mini-univers et structures invisibles

### 5.5.1 Univers simple (S-Type)

Considérons un mini-univers très simple, composé principalement de **cordes** (S) reliées entre elles :


UNIVERSE {
  SIG(3, 0, 0, 1, 1)
  S: "s1" : ()
  S: "s2" : (s1)
  S: "s3" : (s1, s2)
}


Pour un observateur “interne” à ce mini-univers, seuls certains motifs d’interaction seront visibles.  
La structure exacte des connexions entre cordes — et a fortiori la signature globale — restera **hors-champ**, même si elle conditionne les phénomènes observés.

### 5.5.2 Univers avec dimensions compactifiées (C-Type)

Un univers avec des **dimensions compactifiées** peut être caractérisé par :


UNIVERSE {
  SIG(2, 1, 2, 3, 0)
  S: "s1" : (C1)
  S: "s2" : (C2)
  B: "brane" : (s1, s2)
  C: "d1" : ()
  C: "d2" : ()
}


L’observateur ne perçoit que les **dimensions effectives** (\(Deff = 3\)) et les phénomènes qui s’y déploient.  
Les dimensions compactifiées (\(Nc = 2\)) restent **ontologiquement présentes**, mais **phénoménalement hors-champ**.

Dans la TUHC, cet exemple illustre :

- comment un hors-champ ontologique peut être **mathématiquement décrit**,  
- tout en restant **perceptivement inatteignable** pour les acteurs situés dans l’univers.

### 5.5.3 Univers intriqué (E-Type) et structures sociales

En introduisant des morphs **E** (intrication), on peut représenter des univers où :

- des éléments spatialement distincts sont structurellement corrélés,  
- des phénomènes “à distance” partagent un fond commun invisible.

On peut alors faire le parallèle avec des **structures sociales** :

- des acteurs éloignés dans le champ visible (par exemple, dans différents pays, institutions, plateformes) peuvent être intriqués par des **liens hors-champ** (intérêts partagés, dépendances économiques, architectures techniques communes),  
- ces liens ne sont pas toujours explicitement représentés, mais ils structurent les **possibles politiques**.

L’axe ontologique permet ainsi de **relier** :

- la description formelle de mini-univers MULS,  
- et l’analyse des hors-champs sociaux et esthétiques développés dans les chapitres précédents.

Ce chapitre a montré comment l’axe ontologique de Sankamba, articulé à MULS, permet de penser le hors-champ comme **structure fondamentale** plutôt que simple marge du visible.  
Les chapitres suivants proposeront une **synthèse** de ces trois axes (politique, esthétique, ontologique) et développeront des **applications pratiques** où la TUHC devient un outil opératoire d’analyse, de conception et de simulation.
